# Silver Stays

**Tu nueva etapa en el paraíso.**

Silver Stays is a premium platform designed to facilitate the transition of seniors to the Costa del Sol (Spain), providing a comprehensive ecosystem for housing, health, and community.

## 🚀 Project Status
- **Current Phase:** Minimum Viable Product (MVP) - Landing Page
- **Tech Stack:** Next.js 16, TypeScript, Tailwind CSS, Shadcn/ui

## 🧩 Key Components
The application is structured around a modular landing page (`src/app/page.tsx`):

1.  **Hero Section:** Introduces the value proposition with a high-impact visual of Marbella.
2.  **Steps Section:** Outlines the 3-step process (Questionnaire -> Manager -> logistics).
3.  **Trust Bar:** Establish credibility.
4.  **Bento Grid:** Highlights key benefits (Verified Housing, Medical Support, Community).
5.  **Resource Center:** Provides educational content (implied).

## 🛠 Setup & Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
# Access at http://localhost:3000
```
